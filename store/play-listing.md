## Full description (max 4000)

### EN
Rybbit analytics (Unofficial) is a lightweight, open-source mobile client for Rybbit — the privacy-friendly, cookieless web analytics platform. View your traffic, sessions, events and errors on the go from your own Rybbit instance.

This is an UNOFFICIAL, community-built app. It is NOT affiliated with, endorsed by, or supported by the Rybbit team. "Rybbit" is a trademark of its respective owner; this app connects to your Rybbit server only.

Features:
• Real-time dashboard — live users, sessions, pageviews, bounce rate, duration
• Time-series charts with period-over-period comparison
• Sessions list with device, browser, country and event details
• Event log with searchable, structured event properties
• Errors, Locations, Journeys, Retention, Funnels, Goals, Performance
• Web and mobile-app site types
• Multiple organizations and sites
• Light and dark theme, 10+ languages

Connecting:
• Self-hosted: enter your server URL and sign in with email/password or an API key.
• Rybbit Cloud: connect with an API key (available on Standard/Pro plans).

Open source (AGPL-3.0). Your credentials stay on your device; the app talks only to the Rybbit server you configure.

## Data safety
- Collected/stored on device: server URL, email, password or API key (in Android secure storage / SharedPreferences fallback). Not shared with third parties.
- Network: app sends requests only to the user-configured Rybbit server.
- Crash reporting: Sentry (optional, only if SENTRY_DSN configured at build). Diagnostics only.
- No advertising, no third-party analytics SDK.
